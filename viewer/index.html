Help

Using SpyCord

Server Rail

The far-left rail shows available servers. Unread servers get a red dot, and on mobile the rail opens as a larger left-side drawer for easier tapping.

Channel Browser

The channel browser lists channels for the selected server only. Use the three-dot menu to add a server to monitoring or choose exactly which channels should be monitored.

Settings

Settings stores your bot token, optional Discord webhook URL, and highlight keywords. Save and Start reconnects the monitor using those saved values.

Search

Search filters the current message view by text, usernames, channels, categories, and event labels. On mobile it expands inline from the header and can be cancelled with the X button.

Highlights

The Highlights button switches the current channel into a filtered view that only shows mentions of the monitored user and messages matching your saved keywords.

Webhook Alerts

When webhook alerts are enabled for monitored channels, new matching channel messages are posted to your Discord webhook with an embed containing content and metadata.

Live Updates

SpyCord refreshes automatically. The header timestamp shows the last refresh, and loading indicators appear when switching into filtered highlight views.
