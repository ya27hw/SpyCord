Using SpyCord

Server Rail

The far-left column shows the servers this bot can monitor. Click a server icon to switch the message view to that server.

Channel Browser

The middle sidebar lists only the channels for the selected server. Use the edge button to collapse it for more room.

Settings

Use the gear icon to save your bot token and choose which discovered servers should actually be monitored.

Search

The search bar filters the current view by message text, usernames, channels, categories, and event labels.

Mentions

Messages that mention the monitored account are highlighted in yellow, similar to Discord mention highlights.
